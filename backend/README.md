# LeaseLens AI Backend Service

AI-powered lease financial workflow automation.

## Tech Stack
- FastAPI
- SQLAlchemy (async)
- PostgreSQL + pgvector
- Poetry (Package Management)
- Ruff, Black, Mypy (Linting & Formatting)
- Pytest (Testing)

## Getting Started
1. `poetry install`
2. `poetry shell`
3. `uvicorn app.main:app --reload`
